AWPL Leader Landing (Separated Files)
====================================
Files:
- index.html
- styles.css
- main.js
- leader-photo.jpg (placeholder)

How to use:
1) Open main.js and set WHATSAPP_NUMBER to the full number with country code.
   Example for India: 919760085110
2) Open index.html and replace [Leader Name], [X], and meta description text.
3) Replace leader-photo.jpg with the real photo (same file name), or update the <img> src.
4) Optionally host on Netlify, Vercel, GitHub Pages, or your own hosting.

WhatsApp Link Format:
https://wa.me/<countrycode+number>
Example: https://wa.me/919760085110

Tip: Keep the number without + or spaces.
